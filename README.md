# Fake News Detection System

**Tracking and Tracing of Fake News using Machine Learning and AI Verification**

A production-quality Flask web application that detects fake news using:
- **Primary layer**: Scikit-learn Passive Aggressive Classifier + TF-IDF
- **Secondary layer**: Perplexity AI fact-verification API

## Setup Instructions

### 1. Install Python dependencies

```bash
pip install -r requirements.txt
```

### 2. Download NLTK data (one-time, automatic during training)

```bash
python -c "import nltk; nltk.download('stopwords')"
```

### 3. Place dataset files

Place `true.csv` and `fake.csv` in the project root.

Expected columns: `title`, `text` (at minimum `text` is required).

**Recommended dataset**: [LIAR dataset](https://www.cs.ucsb.edu/~william/data/liar_dataset.zip) or [Kaggle Fake News Dataset](https://www.kaggle.com/c/fake-news/data)

### 4. Train the model

```bash
python train_model.py
```

This will print accuracy score, classification report, and save:
- `model/model.pkl`
- `model/vectorizer.pkl`

### 5. Configure API key

Edit `.env` and set your Perplexity API key:

```env
PERPLEXITY_API_KEY=your_actual_key_here
```

Get your key from: https://www.perplexity.ai/settings/api

### 6. Run the application

```bash
python app.py
```

Then open: **http://127.0.0.1:5000**

Admin panel: **http://127.0.0.1:5000/admin**

## Decision Logic

| ML Result | API Result   | Final Verdict                              |
|-----------|--------------|---------------------------------------------|
| Real      | Verified     | ✅ Verified Real News                        |
| Fake      | Not Verified | 🚫 Fake News Confirmed                       |
| Real      | Not Verified | ⚠️ Conflicting — Manual Verification Needed  |
| Fake      | Verified     | ⚠️ Conflicting — Manual Verification Needed  |
| Any       | API Failed   | ML result only (graceful fallback)           |

## API Endpoints

| Method | Path         | Description                        |
|--------|--------------|------------------------------------|
| GET    | `/`          | Main detection UI                  |
| POST   | `/predict`   | JSON prediction endpoint           |
| GET    | `/admin`     | Admin dashboard                    |
| GET    | `/api/logs`  | JSON log data for admin            |
| GET    | `/health`    | Health check + model status        |

### POST /predict — Request Body

```json
{
  "text": "Your news article text here..."
}
```

### POST /predict — Response

```json
{
  "ml_label": "real",
  "ml_confidence": 87.4,
  "api_status": "verified",
  "api_explanation": "VERIFIED. The claim about...",
  "verdict": "✅ Verified Real News",
  "verdict_class": "success",
  "api_used": true
}
```

## Tech Stack

- **Backend**: Python 3.10+, Flask 3.x

- **Text Processing**: NLTK stopwords
- **API**: Perplexity AI (sonar model)
- **Frontend**: HTML5, vanilla CSS, vanilla JavaScript
- **Config**: python-dotenv
